How can we help?

Find answers to common questions about the PyNexus Launcher.

The Launcher says "Server Offline". What do I do?

This usually means the central Game Store API is undergoing maintenance or your internet connection is blocked by a firewall. Check your connection or try restarting the launcher as an Administrator.

A Unity game won't start after downloading.

Ensure you have the DirectX End-User Runtimes and Visual C++ Redistributable installed. If the game crashes immediately, try deleting the game folder in /installed_games/ and clicking "Download" again.

Windows Defender flagged the Launcher as a virus.

Because the launcher is an "unrecognized" app compiled with PyInstaller, Windows may flag it. You can safely click "More Info" and "Run Anyway." As an open-source project, you can always inspect the code on our GitHub.

Where are my games installed?

By default, all games are downloaded into the installed_games directory located in the same folder as your launcher.exe.

Will there be any native modding support?

For now, no. Unfortunately, there is no easy way to implement a modloader solution for EVERY possible game we host, not do I have the willpower to do so. It will be implemented later in development but as of right now, no.
